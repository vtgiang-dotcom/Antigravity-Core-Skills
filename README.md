# Antigravity Core Agentic Skills

This repository contains a collection of foundational, professional-grade skills for the **Antigravity AI Assistant**. These skills are designed to elevate the AI from a standard coding assistant into a disciplined, safety-first Senior Engineer.

## 📦 What's Inside?

| Skill | Purpose |
|-------|---------|
| **[Code Review Expert](./code_review_expert/SKILL.md)** | Enterprise-grade code reviews with a rigorous 3-phase security audit and vulnerability confidence scoring. |
| **[Git Workflow Master](./git_workflow_master/SKILL.md)** | Standardized git management focusing on safety protocols, secret scanning, and automated PR generation. |
| **[File Editor Pro](./file_editor_pro/SKILL.md)** | Precise, non-destructive file modification techniques with stale-read protection and target content isolation. |

## 🚀 How to Use

To use these skills with Antigravity:

1. Copy the skill folders into your `.agent/skills/` or global `~/.gemini/antigravity/skills/` directory.
2. In your conversation with Antigravity, invoke them by name, e.g., `@[code_review_expert] scan this file`.
3. The AI will automatically adopt the professional patterns and strict safety protocols defined in these skills.

## 💡 Inspiration: The Claude Code Paradigm

This project was born out of deep architectural research into top-tier agentic systems, heavily inspired by the routing, tool safety, and cognitive constraints seen in **Claude Code**. 

While others have attempted to run or port the actual backend engines, **Antigravity Core Skills** takes a different approach: we extracted the *underlying engineering discipline* and transformed it into universal Markdown prompt files. This allows any AI Assistant to adopt Claude Code's rigorous standards—such as 3-phase security audits, stale-read file protection, and strict git safety—completely natively, without needing custom binaries.
